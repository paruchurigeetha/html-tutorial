Bootstrap Components

Bootstrap provides ready-made UI components like Accordions, Cards, Carousel and List groups. These components help us build modern websites quickly without writing complex CSS.

📌 Components Covered

  • Accordion → Expand/Collapse content
  • Cards → Display content blocks
  • Carousel → Image slider
  • List Group → Structured lists
