HTML Forms

📖 What is HTML Form?

HTML forms are used to collect user input from the user. These inputs can include text, email, password, selections, etc. Forms are widely used in login pages, registration pages, feedback forms and surveys.

📌 Important Form Elements

  • <input> → Used to take input (text, email, password)
  • <label> → Describes input fields
  • <select> → Dropdown list
  • <textarea> → Multi-line input
  • <button> → Submit button

📌 Syntax

<form>
  <input type="text">
  <input type="email">
  <button>Submit</button>
</form>

📌 Types of Input

  • text
  • email
  • password
  • radio
  • checkbox
