CSS Introduction

📖 What is CSS?

CSS (Cascading Style Sheets) is used to style and design HTML elements. It controls colors, fonts, layouts, spacing and overall appearance of a webpage.

📌 Types of CSS

  • Inline CSS → inside tag
  • Internal CSS → inside <style>
  • External CSS → separate file

📌 Syntax

selector {
   property: value;
}

📌 Example

p {
   color: red;
}
