HTML Tables

📖 What is HTML Table?

HTML tables are used to display data in a structured format using rows and columns. Tables are commonly used for showing student details, marks, reports, and schedules.

📌 Important Tags

  • <table> → Defines table
  • <tr> → Table row
  • <th> → Table header
  • <td> → Table data

📌 Syntax

<table>
  <tr>
    <th>Name</th>
    <th>Branch</th>
  </tr>

  <tr>
    <td>Geetha</td>
    <td>CSE</td>
  </tr>
</table>

📌 Features of Tables

  • Organizes data clearly
  • Supports borders and styling
  • Can merge cells using colspan & rowspan
