HTML Introduction

📖 What is HTML?

HTML (HyperText Markup Language) is the standard language used to create web pages. It provides structure to the content using elements like headings, paragraphs, images, and links. Every website you see on the internet is built using HTML.

📌 Features of HTML

  • Simple and easy to learn
  • Platform independent
  • Supports multimedia (images, videos)
  • Used with CSS and JavaScript

📌 Basic Structure

<!DOCTYPE html>
<html>
<head>
    <title>Page</title>
</head>
<body>

Content goes here

</body>
</html>

📌 HTML Elements

An HTML element consists of opening tag, content, and closing tag. 

<p>This is paragraph</p>

📌 Common Tags

  • <h1> to <h6> → Headings
  • <p> → Paragraph
  • <br> → Line break
  • <hr> → Horizontal line
