CSS Text Styling

📖 Introduction

CSS text properties are used to control the appearance of text content on a webpage. They help in improving readability, alignment, spacing and overall design. Using proper text styling makes the UI look professional and user-friendly.

📌 Important Properties

  • text-align → left, center, right
  • text-decoration → underline, line-through
  • text-transform → uppercase, lowercase, capitalize
  • letter-spacing → space between letters
  • line-height → space between lines
  • font-size → text size
  • font-weight → bold text

📌 Syntax

p {
  text-align: center;
  text-decoration: underline;
  font-size: 20px;
}

📌 Real Usage

  • Headings alignment
  • Highlighting important content
  • Designing professional UI text
