HTML Images

📖 What are Images?

Images are used to display pictures on web pages. They improve user experience and make the website visually attractive. HTML uses the <img> tag to insert images.

📌 Syntax

<img src="image.jpg" alt="description">

📌 Attributes

  • src → Path of image
  • alt → Alternative text
  • width / height → Size of image

📌 Uses

  • Show product images
  • Create gallery
  • Add profile pictures
