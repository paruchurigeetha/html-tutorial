CSS Colors

📖 Introduction

CSS colors are used to define the visual appearance of text, backgrounds, borders and other elements. Using colors effectively improves user experience and readability.

📌 Types of Colors

  • Color Names → red, blue, green
  • HEX Values → #ff0000
  • RGB Values → rgb(255,0,0)
  • RGBA Values → rgba(255,0,0,0.5)

📌 Syntax

p {
  color: red;
  background-color: yellow;
}

📌 Applications

  • Text styling
  • Background design
  • Buttons and UI elements
