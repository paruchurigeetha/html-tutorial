CSS Box Model

The CSS Box Model is used to define the layout and spacing of elements. Every element is treated as a rectangular box consisting of content, padding, border, and margin.

📌 Components

  • Content → actual text or image
  • Padding → space inside border
  • Border → outline
  • Margin → space outside
div {
  margin:20px;
  padding:10px;
  border:2px solid black;
}
