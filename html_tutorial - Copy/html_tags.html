HTML Tags

📖 What are Tags?

HTML tags are used to define elements in a webpage. They tell the browser how to display content. Most tags come in pairs: opening and closing tag.

📌 Types of Tags

  • Paired Tags → <p>...</p>
  • Empty Tags → <br>, <img>

📌 Syntax

<tagname> Content </tagname>

📌 Common Tags

  • <p> Paragraph
  • <a> Link
  • <img> Image
  • <ul> List
