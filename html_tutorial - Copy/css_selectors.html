CSS Selectors

📖 What are Selectors?

CSS selectors are used to select and target HTML elements so that styles can be applied to them. Selectors define "which element" you want to style.

📌 Types of Selectors

  • Element Selector → selects all elements (p, h1)
  • Class Selector → selects elements with class (.box)
  • ID Selector → selects unique element (#id)
  • Group Selector → selects multiple elements
  • Universal Selector → selects all elements (*)

📌 Syntax

p {
   color: red;
}

.box {
   background: blue;
}

#id {
   font-size: 20px;
}

📌 Why Selectors are Important?

  • Helps apply styles efficiently
  • Improves code readability
  • Allows reuse of styles
