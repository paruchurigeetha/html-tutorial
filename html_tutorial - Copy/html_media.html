HTML Media

📖 What is HTML Media?

HTML Media refers to adding multimedia content such as videos and audio files into a web page. It enhances user experience by making websites more interactive and engaging.

📌 Media Elements

  • <video> → Used to embed video files
  • <audio> → Used to embed audio files

📌 Syntax

<video controls>
  <source src="video.mp4">
</video>

<audio controls>
  <source src="audio.mp3">
</audio>

📌 Attributes

  • controls → Shows play/pause buttons
  • autoplay → Plays automatically
  • loop → Repeats media
  • muted → Starts muted

📌 Uses

  • Online learning videos
  • Music players
  • Product demos
