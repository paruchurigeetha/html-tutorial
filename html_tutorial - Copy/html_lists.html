HTML Lists

📖 What are HTML Lists?

HTML lists are used to group related items in a structured format. They make content more readable and organized. Lists are commonly used in menus, instructions, and data display.

📌 Types of Lists

  • Ordered List (ol) → Numbered list
  • Unordered List (ul) → Bulleted list
  • Description List (dl) → Terms and descriptions

📌 Syntax

<ul>
  <li>Item</li>
</ul>

<ol>
  <li>Item</li>
</ol>

📌 Features

  • Improves readability
  • Easy to style using CSS
  • Used in navigation menus
