HTML Links

📖 What are HTML Links?

HTML links are used to connect one webpage to another webpage. They allow users to navigate easily between pages and websites. Links are created using the <a> (anchor) tag.

📌 Syntax

<a href="url">Link Text</a>

📌 Types of Links

  • External Link → Opens another website
  • Internal Link → Opens another page in same website
  • Email Link → Opens email

📌 Attributes

  • href → URL of page
  • target="_blank" → Opens in new tab
